# Musician Booker

## Technologies Used

- Next.js
- Tailwind CSS for efficient styling and organized components
- Prisma (ORM)
- SQLite
- React Query
- Axios
- React Hook Form
- DaisyUI

## Getting Started

1. `npm install` to install dependencies
2. `npx prisma migrate dev --name init` to create the database
2. `npm run seed` to insert initial data
